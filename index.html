MineMon

This is the MineMon download page. MineMon is an App and PC Program that connect together so that your phone will show whatever is on your PC Screen

This page gives you the PC Program. Please download the .jar, and run by double-clicking it !!!

You can get the Mobile app on Google Play After opening the app, enter the code displayed into the box, and press connect.

Troubleshooting:

If when you type the code in, and it says 'Not Connected', there are 3 reasons to why this might happen:

  1. You typed the code incorrectly. Try to type the code in again.
  2. Your phone and computer are on different networks. Make sure they are on the same network (e.g. connected to the same broadband router).
  3. Another computer is 'locked' into the phone. This is a feature to prevent interference. Close the program on the other computer, and try again.
