Troubleshooting:

If when you type the code in, and it says 'Not Connected', there are 3 reasons to why this might happen:

1. You typed the code incorrectly

Try to type the code in again.

2. Your phone and computer are on different networks

Make sure they are on the same network

3. Another computer is 'locked' into the phone

This is a feature to prevent interference. Close the program on the other computer, and try again.
